This project is a complete user authentication system developed using PHP, MySQL, HTML, CSS, and JavaScript. It includes user registration, login, logout, session-based access control, and password recovery. The UI follows a modern glassmorphism design and is fully responsive. Ideal for beginners learning backend authentication and database integration.
